Первичная настройка, необходимая для работы скрипта:
1) В config.py заполните api_id и api_hash (получить на my.telegram.org) и admins (там напишите id своего основного аккаунта, с ним ничего не случится, просто аккаунт юзербота будет принимать команды от Вашего основного аккаунта), если нужен прокси, то заполните соответсвующие поля, комментарии все объяснят.
2) В chats.txt напишите чаты, опять же комментарии все разъяснят, но в принципе чаты в том формате, что Вы присылали (решетки не нужны, их наличие объяснено в комментариях)


При первом запуске:
1) Убедитесь, что на устройстве устновлен Python 3.7.4, если не установлен, то установите.
2) Откройте cmd (Windows) или терминал (Linux)
3) Введите cd <путь до папки>
4) Введите python -m venv venv

Для Linux:
5) Введите source venv/bin/activate
Для Windows:
5) Введите venv\Scripts\activate

6) Введите pip install -r requirements.txt
7) Введите python userbot.py
8) Войдите в аккаунт, номер должен быть в международном формате.



Запуск на Linux:
1) откройте терминал
2) Введите cd <путь до папки>/venv
3) Введите . bin/activate
4) Введите cd <путь до папки>
5) Введите python userbot.py

Запуск на Windows
1) откройте cmd
2) Введите cd <путь до папки>
3) Введите venv\Scripts\activate
4) Введите python userbot.py


Использование:
1) Напишите сообщение для рассылки юзерботу в лс
2) Напишите команду /mailing реплаем на сообщение.
